# AR-Info
Robi/Airtel Number To Location And Information 


### Installation :

```bash
pkg update && pkg upgrade -y
pkg install git -y
pkg install python ruby -y
gem install lolcat
git clone https://github.com/ShTasrif/AR-Info
pip install requests halo
cd AR-Info
python arinfo.py
```
### Run Tools

```bash
arinfo
```
